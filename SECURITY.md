# Security Policy

## Supported Versions

Only the latest version of the Failure Archive is currently supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please report it privately.

Please do **not** open a public issue on GitHub or Swecha. Instead, send an email to security@failure-archive.intelligence (Placeholder).

Include a detailed description of the vulnerability and steps to reproduce it. We aim to respond within 48 hours and provide a fix as soon as possible.
